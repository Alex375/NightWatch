This package contains the model of an Anglehead Flashlight with realistic PBR materials. 
The model presented in three colors.

Package components: 

One mesh: 
Anglehead Flashlight - 980 tris, 
 
Six textures 2048x2048 TGA:
- 3 Albedo Map
- 1 Normal Map
- 1 Metallic-Smoothness Map
- 1 Emission Map

You able to use Emission Map value to switch flashlight on and off.
Installation is not required. Just import the package, grab a necessary prefabs, and use it in your game.
